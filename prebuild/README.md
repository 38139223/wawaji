1.安卓板上运行的应用  (仅限雪暴娃娃机配套主板，安卓板固件请升级至1221版本或以上，安卓系统版本4.4)
包含功能：
1.1通用RTMP的低延迟推流
1.2开机自启动（留意第一次安装后，要启动应用，开启5分钟后方可关闭，
               以保证开启自启成功注册）
1.3与娃娃机的指令交互
1.4与指令服务器的交互
1.5与配置服务器的交互
1.6与局域网配置工具的交互
1.7断流重推

2.局域网配置工具
2.1查找局域网内的安卓板（启动了我们的应用的安卓板）
2.2与安卓板交互，实现局域网配置
2.3通用RTMP流低延迟播放功能

3.公网配置工具
3.1与配置服务器的交互，实现公网配置自己帐号下的安卓板
3.2通用RTMP流低延迟播放功能

4.指令服务器 （默认端口  对于安卓板是7770，对于玩家手机客户端是7771）
4.1监听端口，安卓板上的应用启动后，
   会自动注册到指令服务器，玩家客户端可通过指令服务操作娃娃机

5.配置服务器  （默认端口 对于安卓板是7776，对于公网配置工具是7778）
5.1监听端口，安卓板的应用启动后，
   会自动注册到配置服务器，外网配置工具可通过配置服务器配置安卓板参数
5.2目前没做用户认证
